# Sistema Acadêmico

## Academy
Academy é um sistema criado para o projeto interdisciplinar do curso de Desenvolvimento de Software Multiplataforma, focado em um sistema educacional

## Diretoria
- Publicar eventos (OWR)
- Publicar vagas (OWR)
- Publicar notícias instituicionas (OWR)
- Gerenciar matrículas
- Gerenciar turmas e horários
- Gerar relatórios acadêmicos
- Criar matérias
- Gerir calendário acadêmico
- Gerir funcionários

## Professor
- Lançar notas
- Lançar faltas
- Observação do aluno
- Lançar atividade (e atribuir nota)
- Criar plano de aula
- Disponibilizar material didático
- Criar e gerenciar fóruns de discussão

## Aluno
- Adicionar livros
- Visualizar notas e histórico acadêmico
- Visualizar faltas (validação de limite)
- Solicitar revisão (notas e faltas)
- Solicitar documentos
- Eventos (homepage com próximas atividades, eventos, etc.)
- Acessar material didático
- Participar de fóruns de discussão
- Enviar trabalhos e atividades
- Acompanhar progresso acadêmico
- Biblioteca virtual
- Visualizar grade horaria
