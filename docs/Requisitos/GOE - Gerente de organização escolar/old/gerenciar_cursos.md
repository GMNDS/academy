### Gerenciamento de Cursos

#### Fluxo normal

1. Abrir o sistema acadêmico  
2. Logar na conta de ADM (ou Gestor)  
3. Entrar na aba de “Recursos”  
4. Selecionar a opção “Cursos”  
5. Selecionar o curso o qual deseja consultar  
6. Selecionar o semestre   
7. Selecionar a opção “Alunos”  
8. Pesquisar pelo aluno com base no nome ou disciplina

#### Fluxo de exceção

5.1 - Caso o aluno tenha reprovado em alguma matéria ele pode se encontrar em mais de um semestre distinto  
7.1 - Caso o aluno não esteja mais matriculado, tenha concluído ou trancado o curso, este não aparecerá na pesquisa (pois esta é usada como base o semestre atual)
