## Publicar Eventos
### Usuários
- Administração
- Diretoria

### Fluxo Normal
1. Efetuar login com autorização cabível (Administração ou Diretoria). <!-- Interface do usuário -->
2. Acessar o aplicativo. <!-- Interface do usuário -->
3. Acessar eventos. <!-- Interação do sistema -->
4. Clicar em adicionar eventos. <!-- Interface do usuário -->
5. Escrever novo evento com título. <!-- Interface do usuário -->
6. Escrever evento, podendo adicionar anexos (como imagens e vídeos). <!-- Interface do usuário -->
7. Selecionar público que poderá visualizar o novo evento. <!-- Interface do usuário -->
8. Clicar em publicar. <!-- Interação do sistema -->

### Fluxo de Exceção
6.1. O novo evento não será publicado enquanto estiver em branco o campo de título e o campo de texto. <!-- Interação do sistema -->
7.1. O novo evento não será publicado enquanto não for selecionado o público. <!-- Interação do sistema -->
