## Adicionar feedback a um aluno

### Usuários
- Usuário deslogado

### Fluxo Normal
1. Usuário abre o portal acadêmico.
2. Usuário clica no botão login
3. Usuário adiciona RA ou e-mail
4. Usuário digita senha
5. Usuário clica em logar
6. Servidor valida se login e senha são válidos
7. Usuário se loga

### Fluxo De exceção
6.1. Se o RA ou e-mail for inválido o usuário receberá a mensagem "Erro - credenciais inválidas"
6.2. Se a senha for inválisa o usuário receberá a mensagem "Erro - credencias inválidas"
6.3. Se o usuário errar a senha 5 vezes seus login será bloqueado por 5 minutos
