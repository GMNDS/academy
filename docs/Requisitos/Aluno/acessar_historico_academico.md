## Acessar Histórico Acadêmico
### Usuários
- Aluno

### Fluxo Normal
1. O aluno abre o portal acadêmico. <!-- Interface do usuário -->
2. O aluno realiza o login na plataforma. <!-- Interface do usuário -->
3. Após abrir a interface, o aluno clica em configurações. <!-- Interface do usuário -->
4. O aluno clica em histórico escolar. <!-- Interface do usuário -->
5. O aluno seleciona, por meio de um dropdown, o semestre desejado. <!-- Interface do usuário -->
6. O sistema exibe as disciplinas, as notas, a carga horária das disciplinas e o resultado final. <!-- Interação do sistema -->

### Fluxo de Exceção
2.1. Caso o sistema demore muito para realizar o login, é exibida uma mensagem de erro: "Tente novamente mais tarde". <!-- Interação do sistema -->
5.1. Caso o aluno selecione um semestre que ainda não foi concluído, a tabela do histórico não será exibida e será notificado: "Semestre ainda não concluído". <!-- Interação do sistema -->
