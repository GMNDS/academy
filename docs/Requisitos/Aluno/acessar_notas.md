## Acessar Notas
### Usuários
- Professor
- Aluno

### Fluxo Normal
1. O usuário abre o portal acadêmico. <!-- Interface do usuário -->
2. O usuário faz login na plataforma. <!-- Interface do usuário -->
3. O usuário acessa o painel de controle. <!-- Interação do sistema -->
4. O usuário navega até a seção "Notas". <!-- Interface do usuário -->
5. O usuário seleciona uma disciplina específica. <!-- Interação do sistema -->
6. O sistema exibe as notas, incluindo avaliações, pesos e médias finais. <!-- Interação do sistema -->

### Fluxo de Exceção
4.1. Caso o professor não tenha notas registradas, será exibida a mensagem "Nenhuma nota disponível para esta disciplina". <!-- Interação do sistema -->
