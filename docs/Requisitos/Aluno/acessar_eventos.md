## Requisito 2: Acessar Eventos
### Usuários
- Administração
- Diretoria
- Alunos

### Fluxo Normal
1. Efetuar login com autorização cabível (Administração, Diretoria ou Alunos). <!-- Interface do usuário -->
2. Acessar o aplicativo. <!-- Interface do usuário -->
3. Acessar o menu. <!-- Interação do sistema -->
4. Acessar eventos. <!-- Interação do sistema -->
5. Visualizar eventos. <!-- Interação do sistema -->

### Fluxo de Exceção
4.1. Caso o sistema não consiga carregar os dados de eventos, será exibida a mensagem "Erro ao carregar os dados de eventos. Tente novamente mais tarde". <!-- Interação do sistema -->
