## Acessar Presença
### Usuários
- Professor
- Aluno

### Fluxo Normal
1. O usuário ou aluno abre o portal acadêmico. <!-- Interface do usuário -->
2. O usuário ou aluno faz login na plataforma. <!-- Interface do usuário -->
3. O usuário ou aluno acessa o painel de controle. <!-- Interação do sistema -->
4. O usuário ou aluno navega até a seção "Presença". <!-- Interface do usuário -->
5. O usuário ou aluno seleciona uma disciplina específica. <!-- Interação do sistema -->
6. O sistema exibe o registro de presença, incluindo datas de aula e status (presente, ausente, justificado). <!-- Interação do sistema -->

### Fluxo de Exceção
4.1. Caso o sistema não consiga carregar os dados de presença, será exibida a mensagem "Erro ao carregar os dados de presença. Tente novamente mais tarde". <!-- Interação do sistema -->
