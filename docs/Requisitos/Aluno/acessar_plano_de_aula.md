## Requisito 1: Acessar Plano de Aula
### Usuários
- Professor
- Aluno

### Fluxo Normal
1. O usuário abre o portal acadêmico. <!-- Interface do usuário -->
2. O usuário faz login na plataforma. <!-- Interface do usuário -->
3. O usuário acessa o painel de controle. <!-- Interação do sistema -->
4. O usuário navega até a seção "Disciplinas". <!-- Interface do usuário -->
5. O usuário seleciona uma disciplina específica. <!-- Interação do sistema -->
6. O sistema exibe o plano de aula da disciplina, incluindo tópicos, atividades e datas de entrega. <!-- Interação do sistema -->

### Fluxo de Exceção
5.1. Caso o sistema não consiga carregar o plano de aula, será exibida a mensagem "Erro ao carregar o plano de aula. Tente novamente mais tarde". <!-- Interação do sistema -->
